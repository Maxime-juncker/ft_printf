## Goals
this simplified version of printf need to supprt:
- %c
- %s
- %p
- %d / %i
- %u
- %x / %X
- %%

## Compile it yourself
first clone the repo
```
git clone https://github.com/Maxime-juncker/ft_printf.git
cd ft_printf
```
then build it using make
```
make
```
